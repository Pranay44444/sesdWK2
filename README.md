# CLI Tool — Command Line Interface

A TypeScript-based CLI tool built with Object-Oriented Programming principles. Provides mathematical operations and API integrations.

## Installation

```bash
npm install
npx tsc        # compile TypeScript
npm link       # make `mycli` available globally
```

## Usage

```bash
mycli <command> [arguments]
# or without linking:
node dist/cli.js <command> [arguments]
```

## Commands

### Utility

```bash
mycli greet Raman          # Hello Raman!
```

### Mathematical Operations

**Basic Arithmetic**

```bash
mycli add 5 3              # Result: 8
mycli subtract 10 4        # Result: 6
mycli multiply 6 7         # Result: 42
mycli divide 20 4          # Result: 5
mycli modulo 17 5          # Result: 2
```

**Advanced Math**

```bash
mycli power 2 8            # Result: 256
mycli sqrt 144             # Result: 12
mycli factorial 5          # Result: 120
```

**Statistics**

```bash
mycli average 10 20 30 40  # Result: 25
mycli max 5 12 8 20 3      # Result: 20
mycli min 5 12 8 20 3      # Result: 3
```

### API Integrations

```bash
mycli github <username>    # GitHub profile info (name, repos, followers, etc.)
mycli joke                 # Random joke
mycli pokemon <name>       # Pokémon name, height, and weight
```

## Architecture

Each command is a class with three key components:

- **`constructor(program)`** — receives the Commander instance
- **`register()`** — defines the command name, arguments, description, and action
- **Action handler** — implements the command logic

`AppRunner` in `core/runner.ts` manages command registration and delegates parsing to Commander.js.
